Laboratory Automation Competition

ホーム リーダーボード 投稿ガイド GitHub

1. 概要

LA-Bench 2025は、実験自動化（Laboratory Automation）における言語モデルとAIエージェントの性能を評価する国際的なコンペティションです。

2. タスクカテゴリー

テキスト

  • 論文読解: 化学論文から
    特定の実験条件や結果の抽出
  • 実験計画: 与えられた目標に対するPDFへの出力

ロボット

  • 軌道生成: 実験器具の配置からロボットアームの軌道をYAML形式で生成

5. データセット

  • Example Dataset: 10問（練習用）
  • Public Test Dataset: 10問（Development Phase用）
  • Private Test Dataset: 10問（Test Phase用）

6. 評価方法

Development Phase

  • LLM部門のみ参加可能
  • リーダーボードで順位を確認可能

Test Phase

  • 第1週：LLM部門の評価期間
  • 第2週：専門家部門
    上位10チームの評価期間

評価指標

  • 正確性
  • 完全性
  • 形式の適切性
  • 実行可能性

7. 参加方法

  1. GitHubリポジトリから必要なコードとデータをダウンロード
  2. 実装コードを作成
  3. 指定フォーマットで提出
  4. Google Colabでの実行とローカル環境での確認

8. 主催・協賛

  • 主催: 東京大学（Science Aid）
  • 協賛: 一般社団法人 実験自動化推進協会（LASA）
